# SlateDB Architecture Diagrams

> Comprehensive visual documentation of SlateDB's cloud-native LSM-tree architecture

## Overview

This repository contains detailed architecture diagrams for [SlateDB](https://github.com/slatedb/slatedb)

I have used claude 4.0 to generate mermaid code as well as images

## Available Diagrams

1. **[System Architecture](diagrams/01-system-architecture.mmd)** - Overall component structure and data flow
2. **[Write Path Flow](diagrams/02-write-path-flow.mmd)** - Write path flow
3. **[Read Path Flow](diagrams/03-read-path-flow.mmd)** - Read path flow

## How to Use

### View/Edit Diagrams
1. **Copy diagram code** from `/diagrams/` folder
2. **Paste into [Mermaid Live](https://mermaid.live)** to view/edit
3. **Modify for your own use cases**
